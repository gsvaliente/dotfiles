# Gabriel's Dotfiles

These are just my dotfiles, mostly inspired by YouTube videos or other git
files found online.

## What is included?

- NeoVim
- Tmux
- Ghostty
- Starship

## Things to note

- Lazy as the package manager
- Nvim-cmp as the completion
- Fzf as the fuzzy finder
- Snacks to help out with many QOF upgrades
